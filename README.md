# MealzApp

A Multi module clean architecture Kotlin based application that fetches meals.

# Used

- Glide
- Retrofit
- Dagger-Hilt
- View Binding
- Onion architecture
- MVVM
- Okhttp
- Mutable/LiveData
- Kotlin Coroutines
- Recyclerview
